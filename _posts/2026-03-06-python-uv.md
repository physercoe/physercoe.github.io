If you've ever waited 30 seconds for `pip install` to resolve dependencies, you'll appreciate [uv](https://github.com/astral-sh/uv). It's a Python package manager written in Rust by the team behind Ruff, and it's genuinely fast — often 10–100x faster than pip.

## Installation

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

After that, restart your shell and confirm it's available:

```bash
uv --version
# uv 0.5.x
```

## Creating a virtual environment

uv replaces both `pip` and `venv`. To start a new project:

```bash
# Create a venv in .venv/
uv venv

# Activate it (same as always)
source .venv/bin/activate
```

You can also target a specific Python version — uv will download it automatically if needed:

```bash
uv venv --python 3.12
```

## Installing packages

Drop-in replacement for `pip install`:

```bash
uv pip install requests
uv pip install -r requirements.txt
uv pip install "fastapi[standard]"
```

The first run fetches packages; subsequent runs hit a local cache so repeated installs are nearly instant.

## Running scripts without a venv

`uv run` creates a temporary isolated environment just for that script:

```bash
uv run script.py
uv run --with requests --with rich script.py
```

Handy for one-off scripts you don't want polluting a shared environment.

## Full project workflow

uv understands `pyproject.toml` and can replace pip + venv + pip-tools:

```bash
uv init my-project
cd my-project

uv add httpx        # adds to pyproject.toml + uv.lock
uv remove httpx
uv sync             # sync env to lockfile
```

Commit `uv.lock` to version control — teammates get identical environments.

## Speed comparison

```bash
# pip (cold cache)
time pip install numpy pandas scikit-learn matplotlib
# real: 1m 14s

# uv (cold cache)
time uv pip install numpy pandas scikit-learn matplotlib
# real: 4.2s

# uv (warm cache — the daily reality)
time uv pip install numpy pandas scikit-learn matplotlib
# real: 0.3s
```

## When NOT to use uv

- Publishing packages to PyPI — Poetry or Flit have better build-system integration
- Complex Conda environments with non-Python binaries — stick with Conda/Mamba

## TL;DR cheatsheet

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh  # install
uv venv                                            # create env
uv pip install <package>                           # install package
uv run script.py                                   # run without activating
uv init / uv add / uv sync                        # full project workflow
```
