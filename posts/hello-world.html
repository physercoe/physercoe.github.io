Hello, World — Getting Started with GitHub Pages

March 5, 2026 · github devops

Every blog needs a first post, so here it is. This site is hosted on GitHub Pages — free, fast, and zero infrastructure to manage.

How it was set up

The repo is named physercoe/physercoe.github.io. GitHub automatically serves any repo with that naming pattern at https://physercoe.github.io. No build step, no CI pipeline — just push HTML and it's live.

Pushing with GitHub CLI

After enabling Pages in the repo settings, deploying is just:

git add .
git commit -m "update post"
git push origin main

The token needs to be embedded in the remote URL since gh's credentials aren't automatically shared with Git over HTTPS. Run this once after cloning: 

git remote set-url origin https://$(gh auth token)@github.com/physercoe/physercoe.github.io.git
